System Architecture

Our components are designed to work seamlessly together

The architecture of the system in the third design shows how the mechanical and electronic parts are going to be integrated in automating the fertility detection of the duck eggs. It is run by a duck egg farmer, where eggs are placed inside a 3D-printed tray that sits in a wooden box and followed by a drawer slider mechanism to facilitate lifts and replenishment. The internal system is composed of Raspberry Pi 5 central processing unit, which is connected with a webcam, which allows capturing the images, a touchscreen LCD that will help display the results, and a relay module which will determine when to turn-on the LED lamps used to guide the eggs. It is an externally powered system which has fans to regulate thermal control as well. Such architecture allows optimal and valid fertility assessment, allowing working in real-time with no user input.

Model Training Result

UI Result
Epoch Training

The UI result demonstrates that the trained YOLO model is capable of detecting and classifying duck eggs into balut and penoy with confidence values consistently above 0.85, indicating strong prediction reliability. The bounding boxes are well-aligned with each egg, suggesting that the detection process is precise and effective in real-time conditions. On the other hand, the training metrics graph shows the convergence of the model across epochs 3 to 11. The box loss, class loss, and DFL loss values decline steadily, reflecting the model’s ability to minimize prediction errors. At the same time, the mAP50 and mAP50-95 scores show a gradual upward trend and eventually stabilize, which indicates that the model has achieved consistent accuracy in distinguishing between balut and penoy. Overall, these results confirm that the system is well-trained, stable, and capable of delivering reliable fertility assessment of duck eggs.

Test and Result

List of Related Results Upon Testing

Efficiency

FertiliDuck significantly outperformed the traditional candling method. In 5-minute tests, it detected up to 150 eggs per batch, while manual inspection averaged only 42 eggs. This proves FertiliDuck is over 3x faster, helping farmers save time and boost productivity.

Speed

FertiliDuck analyzed 25 duck eggs in just 1 second per batch, while the traditional method took 3–5 seconds per egg. This shows that FertiliDuck is drastically faster, making it ideal for high-efficiency operations.

Accuracy

While the traditional candling method demonstrated consistent 100% accuracy across all tests, the Fertiliduck device exhibited high accuracy, ranging from 92% to 100%, indicating strong performance with minor variation.
